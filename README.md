# RAG-MVP

**Proof of Concept** für ein Retrieval-Augmented Generation (RAG)-System.  
Ziel ist die Demonstration der automatisierten Verarbeitung medizinischer PDF-Dokumente und die semantische Zuordnung relevanter Begriffe.  

Das MVP umfasst:

- PDF-Upload via Weboberfläche  
- Automatische Verarbeitung & Mapping mit LLM  
- Download des Ergebnisses als CSV-Datei  
- Temporäre Speicherung (max. 24 h, kein persistenter Speicher)

Projektlaufzeit: 01. Oktober 2025 – 30. Januar 2026
